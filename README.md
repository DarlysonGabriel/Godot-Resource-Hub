# Godot Engine Hub Configs

Um repositório com arquivos `.tres` e `.res` prontos para uso na Godot Engine.

Aqui você encontrará recursos pré-configurados para objetos 2D e 3D, com o objetivo de facilitar a criação e organização dos seus projetos.

## ⚠️ Atenção

Este repositório **não contém sistemas ou scripts prontos**.

Ou seja, você **não encontrará** inventários, IA, movimentação de personagens, interfaces ou qualquer outro sistema programado. O foco deste projeto é disponibilizar apenas configurações reutilizáveis de recursos da Godot.

## O que você encontrará?

Diversos arquivos de configuração que podem ser utilizados como complemento para os seus projetos, como:

* Configurações de `MeshInstance3D`
* Configurações de `WorldEnvironment`
* Materiais (`Material`)
* Configurações de iluminação
* Outros arquivos `.tres` e `.res`

Novos recursos serão adicionados conforme o projeto evoluir.

## Objetivo

O objetivo deste repositório é disponibilizar uma biblioteca de recursos prontos para uso, permitindo que desenvolvedores economizem tempo na configuração de objetos e ambientes dentro da Godot Engine.

## Tecnologias

* Godot Engine
* Arquivos `.tres`
* Arquivos `.res`
