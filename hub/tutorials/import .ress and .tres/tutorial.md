# Inport .ress and.tres

## Como inportar arquivos .res e .tres?

Olá a todos!
Neste tutorial, vou ensinar como inportar seu(s) arquivo(s) de configuração .res e .tres.
Quero que você leia até o final para entender o passo a passo.

### 1: preparação

Antes de de fato começar a inportar seu arquivo você deve conprir algins requisitos:
	* 1: Ter este arquivo compactado em .zip/.rar/.
	* 2: Seu arquivo NÃO PODE ser ter um script em GDScript/Python/CSharp ou qulquer outro.

Caso tenha esses requisitos vamos continuar:

### 2: ordem de pastas:

A ordem das pastas deve ser a mesma, para facilitar o download e upload.

O padrão se seguimento de pastas é:

```
res://
├── src/
│   ├── config/
│   │   ├── whater/ (Exenplo)
│   │   │   ├── textures/
│   │   │   │   └── water.png
│   │   │   └── whater.tres
...
```
Atenção, TODOS OS ARQUIVOS (ou pelo menos alguns) devem estar em englês
